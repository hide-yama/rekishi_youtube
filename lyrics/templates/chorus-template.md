# サビテンプレート

## 基本構成

```
覚えよう覚えよう 日本の歴史
年号と出来事 心に刻もう
古代から現代まで 流れる時代
語呂合わせで 楽々暗記
```

## バリエーション例

### パターンA（受験応援型）
```
がんばれ受験生 歴史を学ぼう
語呂で覚える 重要年号
試験に出るよ 基本の基本
これで合格 間違いなしだ
```

### パターンB（時代讃美型）
```
古代の昔から 続く日本史
先人たちの 歩んだ軌跡
平安 鎌倉 室町 江戸
明治 大正 昭和 平成 令和
```

### パターンC（勉強応援型）
```
一つ覚えれば また次へ
歴史の扉が 開いていく
年号マスター 君になろう
知識の力で 未来を拓け
```

## 使い分け

- **era-songs**: 各時代ごとに適したサビを選択
- **complete-songs**: 全体を通してのメインサビ
- **動画の雰囲気**: 楽しい/真面目/応援などトーンに合わせて調整
