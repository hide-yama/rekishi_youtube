# 詩節テンプレート

## 基本構成（8小節）

```
[年号] [語呂合わせ] [出来事]
[時代背景の説明]
[年号] [語呂合わせ] [出来事]  
[歴史的意義の説明]
```

## 例：

```
こんな(57)金印欲しかった
奴国王が後漢へ
文(23)く(9)ださいね卑弥呼より
邪馬台国から魏の国へ
```

## 作詞のポイント

1. **リズム**: 7-5調または5-7調でテンポよく
2. **韻**: 可能な限り語尾を揃える
3. **ストーリー**: 時代の流れを意識
4. **覚えやすさ**: 繰り返しやフレーズの統一感

## メロディ参考
