# rekishi_youtube プロジェクト要件定義書

## プロジェクト概要
受験勉強 × 音楽 × 映像 × 画像を融合した革新的な教育系YouTubeチャンネル

### 目的
- エンターテインメント性と学習効果を両立させた新しい学習スタイルの確立
- 音楽を通じて暗記の苦痛を解消し、楽しく継続できる学習体験の提供
- 中高生の受験勉強をサポートする高品質なコンテンツの制作

### 背景
- 教育系YouTubeは成長市場だが、音楽×勉強分野は未成熟
- 既存の教育コンテンツは「退屈」「覚えられない」「継続できない」という課題を抱える
- AI技術（Suno等）により、少人数でも高品質な音楽制作が可能になった

### 解決したい課題
- 暗記学習の退屈さ・苦痛
- 学習内容が定着しない問題
- 継続的な学習習慣の形成が困難

## ターゲットユーザー
### 主要ターゲット
- **中学3年生～高校2年生（14-17歳）**
- 受験プレッシャーを感じ始める世代
- YouTube/TikTokネイティブ世代
- 音楽（特にYOASOBI等のJ-POP）に親和性が高い

### 市場規模
- 実質市場規模：約450万人（日本全国中高生×YouTube視聴傾向）

## 主要機能
### 必須機能（MVP） - Phase 1
- [ ] **YOASOBI風J-POP×歴史年号コンテンツ**
  - BPM130-150のキャッチーなメロディ
  - 重要年号を歌詞に組み込んだ楽曲
  - シンプルな歌詞動画（グラデーション背景、歌詞アニメーション）
- [ ] **YouTube/TikTok/Shorts対応フォーマット**
- [ ] **SEO最適化されたタイトル・説明文**

### あったら良い機能 - Phase 2
- [ ] **進化系ラップ×英単語シリーズ**
  - 高度な韻踏みによる記憶定着強化
  - Co.慶応を上回る音楽品質
- [ ] **キャラクター入りストーリー演出**
  - 受験生×歴史人物の対話演出
- [ ] **学習効果測定コンテンツ**
  - 動画後のクイズ・テスト機能

### 将来的な拡張機能 - Phase 3
- [ ] **ジャンル横断展開**
  - 数学：テクノ音楽
  - 理科：ロック音楽
  - 英語：R&B
- [ ] **本格アニメMV化**
- [ ] **ユーザー参加型企画（UGC）**
- [ ] **教育アプリ展開** 

## 技術要件
### 音楽制作
- **Suno AI** - AI音楽生成ツール（メイン）
- **DAW（必要に応じて）** - 細かい編集・ミックス用

### 映像制作
- **動画編集ソフト**
  - Phase 1: シンプルな歌詞動画対応（Premiere Pro/DaVinci Resolve等）
  - Phase 2-3: アニメーション対応ソフト（After Effects等）
- **画像生成AI**
  - Midjourney/Stable Diffusion（イラスト生成用）

### 配信プラットフォーム
- **YouTube** - メインプラットフォーム
- **TikTok** - ショート動画展開
- **Instagram Reels** - 補助的展開

### 分析・管理ツール
- **YouTube Studio** - アナリティクス
- **TikTok Analytics** - パフォーマンス分析
- **このプロジェクト管理ツール** - タスク管理

## 非機能要件
### パフォーマンス
- 動画品質：1080p以上（できれば4K対応）
- 音質：高ビットレート（320kbps以上）
- 投稿頻度：週1-2本の安定投稿

### アクセシビリティ
- 字幕対応（聴覚障害者向け）
- 歌詞の見やすさ（フォントサイズ、コントラスト）

### スケーラビリティ
- コンテンツ制作フローの標準化
- テンプレート化による効率化

## 制約事項
- 少人数体制での運営（初期は1-2名想定）
- AI活用による制作コスト削減が前提
- 著作権・教育的正確性への配慮必須

## スケジュール
### Phase 1（1-3ヶ月目）
- 第1弾：歴史年号×J-POP試作品完成
- YouTube/TikTokチャンネル開設
- 初期3-5本の動画投稿
- 反響分析とチューニング

### Phase 2（4-6ヶ月目）
- 英単語×ラップシリーズ開始
- キャラクター演出の導入
- 週1-2本の安定投稿体制確立

### Phase 3（7ヶ月目以降）
- ジャンル横断展開
- 収益化開始
- チーム体制の拡充検討

## 成功指標
### 短期（3ヶ月）
- チャンネル登録者数：1,000人
- 総再生回数：10万回
- エンゲージメント率：5%以上

### 中期（6ヶ月）
- チャンネル登録者数：1万人
- 月間再生回数：100万回
- 収益化達成

### 長期（1年）
- チャンネル登録者数：10万人
- ブランド認知の確立
- 派生ビジネスの開始

## リスクと対策
### リスク
1. **飽きられるリスク**
   - 対策：ジャンル多様化、ユーザー参加企画
2. **制作負荷の増大**
   - 対策：テンプレート化、外注体制構築
3. **教育的正確性の担保**
   - 対策：専門家監修、ファクトチェック体制
