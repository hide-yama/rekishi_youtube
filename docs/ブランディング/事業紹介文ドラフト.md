# 事業紹介文ドラフト

---

## 1. 事業紹介文（ドラフト）

ちえのわ舎（CHIENOWASHA）は、「知恵がめぐり、心がつながる場所」をコンセプトに、複雑な課題にじっくりと向き合い、協働と探求を大切にするパートナーです。

私たちは、単純な答えでは解けない問いに対して、仲間と知恵を出し合い、ぬくもりとユーモアを持って解決策を紡ぎ出します。

子ども心と大人の知性が交差する温かな拠点として、あなたと共に課題を紐解き、信頼と結束を育みながら、長期的なパートナーシップを築いていきます。

---

## 2. キャッチコピー案

- 知恵がめぐり、心がつながる場所
- じっくり、やさしく、課題を紐解く
- あなたと共に、知恵の輪をつなぐ
- 複雑な問いに、ぬくもりとユーモアで
- 一緒に悩み、一緒に発見するパートナー
