# 使用中の有料ツール棚卸・分類

## 📋 目的
フリーランス開業に向けて、現在使用中の有料ツールを整理し、事業用・プライベート用に分類する。

## 🎯 作業内容
1. 現在契約中の有料サービス・ツールをすべてリストアップ
2. 各ツールの用途・必要性を評価
3. 事業用・プライベート用・共用に分類
4. 不要なツールの解約検討
5. 事業用ツールの経費計上準備

## 📝 分類基準

### 🏢 事業用
- 直接的に業務で使用するツール
- クライアントワークに必要なツール
- 事業運営に必要な管理ツール

### 🏠 プライベート用
- 個人的な趣味・娯楽で使用
- 事業とは無関係なツール

### 🔄 共用
- 事業・プライベート両方で使用
- 按分計算が必要なツール
