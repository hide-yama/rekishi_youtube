## 📝 分類基準

### 🏢 事業用
- 直接的に業務で使用するツール
- クライアントワークに必要なツール
- 事業運営に必要な管理ツール

### 🏠 プライベート用
- 個人的な趣味・娯楽で使用
- 事業とは無関係なツール

### 🔄 共用
- 事業・プライベート両方で使用
- 按分計算が必要なツール
