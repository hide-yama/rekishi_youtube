# 個人事業用リスク一覧

## 📋 目的
個人事業主として事業運営上想定されるリスクを洗い出し、対策の優先度を明確にして継続的に管理する。

## 🎯 リスク評価基準
- **影響度**: 高・中・低（事業への打撃レベル）
- **発生確率**: 高・中・低（発生可能性）
- **優先度**: 高・中・低（対策の緊急度）

---

## 💼 1. エージェント業務リスク（レバテック経由・基盤収入）

### 1-1. 案件間空白期間・契約変更リスク
- **リスク内容**: レバテック経由案件の切り替え時期の空白期間発生、契約終了・更新拒否による一時的な収入停止
- **影響度**: 高
- **発生確率**: 中
- **優先度**: 高
- **対策案**: 
  - [ ] 3-6か月分の生活費・事業資金の蓄積
  - [ ] スキルアップによる付加価値向上（継続的に）
  - [ ] レバテックとの継続的なコミュニケーション・関係維持

### 1-2. エージェント・支払いチェーンリスク
- **リスク内容**: レバテック経営悪化・倒産、3層支払いチェーン（メガバンク→トランス・コスモス→レバテック）での遅延・未払い
- **影響度**: 中
- **発生確率**: 低
- **優先度**: 中
- **対策案**:
  - [ ] レバテックの財務状況・評判の定期確認
  - [ ] 15日サイト分の資金準備（約22万円×1.5か月分）
  - [ ] トランス・コスモス、メガバンクとの関係性確認（安定性評価）

## 🚀 2. 自分事業リスク（スモールビジネス展開）

### 2-1. 初期投資回収不能・赤字継続
- **リスク内容**: スモールビジネス立ち上げ費用の回収不能、長期赤字状態
- **影響度**: 中
- **発生確率**: 高
- **優先度**: 中
- **対策案**:
  - [ ] 低コストで検証可能なMVP（最小限実行可能プロダクト）から開始
  - [ ] 各ビジネスの撤退ライン・期限を事前設定
  - [ ] 初期投資の上限額設定（メイン収入への影響最小化）

### 2-2. 市場ニーズの見誤り・需要不足
- **リスク内容**: 想定していた市場ニーズが存在しない、競合過多
- **影響度**: 中
- **発生確率**: 高
- **優先度**: 中
- **対策案**:
  - [ ] 事前の市場調査・顧客ヒアリング徹底
  - [ ] 小規模テスト販売での需要検証
  - [ ] 既存の成功事例・競合分析の実施

### 2-3. 営業・マーケティングノウハウ不足
- **リスク内容**: 技術はあるが集客・販売スキルが不足、顧客獲得不可
- **影響度**: 高
- **発生確率**: 高
- **優先度**: 高
- **対策案**:
  - [ ] マーケティング・営業手法の体系的学習
  - [ ] SNS・コンテンツマーケティングスキル習得
  - [ ] 既存人脈・紹介ルートの活用

### 2-4. 時間配分・本業との両立困難
- **リスク内容**: スモールビジネスに時間を取られ、メイン収入に悪影響
- **影響度**: 高
- **発生確率**: 中
- **優先度**: 高
- **対策案**:
  - [ ] 週単位での時間配分計画策定（メイン業務優先）
  - [ ] 自動化・効率化ツール積極活用
  - [ ] 成果が出ないビジネスの早期撤退判断

### 2-5. ブランディング・信頼構築の困難
- **リスク内容**: 個人事業主として信頼獲得が困難、大手競合に勝てない
- **影響度**: 中
- **発生確率**: 中
- **優先度**: 中
- **対策案**:
  - [ ] 専門分野での実績・事例蓄積
  - [ ] 顧客満足度重視の丁寧なサービス提供
  - [ ] 口コミ・紹介を促進する仕組み構築

### 3-4. 税務ペナルティ・申告漏れ
- **リスク内容**: 確定申告ミス、税務調査、追徴課税、経費処理間違いによる過少申告・過大申告
- **影響度**: 中
- **発生確率**: 中
- **優先度**: 高
- **対象**: 【エージェント業務・自分事業 共通】
- **具体的リスク要因**:
  - 経費と私的支出の混同・区分間違い
  - 事業用と個人用の按分計算ミス（家賃、光熱費、通信費等）
  - 源泉徴収票・支払調書の漏れ・誤記載
  - 消費税課税事業者の判定・処理ミス（売上1000万円超）
  - 青色申告特別控除の要件不備
  - 減価償却資産の処理間違い（PC、車両等）
  - 帳簿記録の不備・根拠書類保存不足
- **対策案**:
  - [ ] 会計ソフト（freee）導入・設定完了
  - [ ] 税理士との相談体制構築（年間契約検討）
  - [ ] 個人事業主向け税務セミナー受講
  - [ ] 経費処理ルールの明文化・チェックリスト作成
  - [ ] 月次での帳簿確認・税理士レビュー体制
  - [ ] 事業用・個人用支出の明確な分離管理
  - [ ] 領収書・請求書の適切な保存システム構築
  - [ ] 確定申告前の税理士による事前チェック必須化
  - [ ] 消費税課税事業者移行時の事前準備・相談
