# 情報セキュリティハンドブック 詳細ブリーフィング（Ver 2.5 最新版）

## 概要
本ブリーフィングドキュメントは、「情報セキュリティハンドブック Ver 2.5（最新版）」をレビューし、その主要テーマ、重要なアイデア、事実を詳細にまとめたものです。
レバテックを通じて業務を行うすべての作業従事者に対し、情報漏洩、改ざん、または利用不能な状態が発生し、望ましくない影響が及ぶ「情報セキュリティ事故」を未然に防ぐための具体的なガイドラインと報告体制を示します。

---

## 主要テーマと重要なアイデア・事実

### 1. 守秘義務と情報セキュリティ事故の定義
- **守秘義務の強調**：作業従事者は、レバテックとの間で締結された「機密保持契約」に基づき、守秘義務があることに合意しています。ハンドブックの規則を遵守し、情報セキュリティ事故の発生を未然に防ぐよう協力が求められています。
- **情報セキュリティ事故の明確な定義**：「情報の『漏えい』『改ざん』の発生、または『利用できない』状態になったときに、当社の業務や顧客、取引先、株主、（個人情報流出等の場合）被害に遭った全関係者に望ましくない影響が及ぶ事」と定義されています。
- **偶発的な事故も対象**：現場からの貸与物の紛失や破損、SNSへの掲載など、「うっかりやってしまったこと」も情報セキュリティ事故となり得ることが明記されています。
- **自己判断の禁止と報告義務**：事故が発生した場合、作業従事者独自の判断ではなく、必ず現場社員やレバテック担当者への報告が義務付けられています。具体的な事故事例は別途資料で提供されます。

### 2. 情報セキュリティ事故発生時の迅速な報告と対応フロー
- **即時報告の徹底**：情報セキュリティ事故が発覚した場合、直ちに参画先企業とレバテックへの報告が必須です。
- **レバテックへの緊急連絡先**：土日祝日問わず、レバテック代表電話（03-5774-1762）へ電話し、事故発生の旨を伝える必要があります。担当営業が不在の場合は、メールでの報告も求められます。
- **事態収束までの状況連携**：事故が収束するまで、参画先企業とレバテックへの迅速かつ細やかな状況連携が求められます。レバテックからも連携を取り、対応依頼を行う場合があるため、必ず対応する必要があります。
- **再発防止策の実施**：事故収束後も再発防止に十分留意し、参画先企業やレバテックと相談の上決定された具体的な再発防止策は必ず実施する必要があります。

### 3. 業務中のインターネット利用に関するルール
- **ウェブサイト利用の注意**：不審なサイトや業務上不要なサイトへのアクセス、現場支給メールアドレスの登録は禁止。
- **パスワードのブラウザ保存は禁止。**
- **オンラインストレージ・共有プラットフォーム利用の注意**：利用には「現場社員のリーダー」の許可が必要。「同じ現場のメンバー以外」との業務関連情報の共有は禁止。
- **メールアドレス登録が必要な場合**：現場社員のリーダーに確認し、現場支給のメールアドレスを利用（個人アカウントの利用は禁止）。
- **業務中のSNS利用の注意**：現場からの依頼以外での業務中のSNS利用は禁止。現場PCでのSNS利用は「現場社員のリーダー」の許可が必要。レバテックや現場の情報の書き込みは禁止。現場社員やメンバーとの私的なSNS交流は、双方の立場をわきまえ、社会人としての良識の範囲で行う。セキュリティ設定を行い、アカウント乗っ取り、なりすましに注意。スマートフォン・タブレット端末上のデータ、写真、位置情報が予期せず公開される可能性に注意。

### 4. 業務中の電子メール送信に関するルール
- **メール送付先の注意**：メールの利用は基本的に現場内に留め、外部への送付は「現場社員のリーダーの許可」を得てから行う。
- **現場支給のメールアドレスで受信したメールを個人メールアドレスに転送することは禁止。**
- **複数の外部に同時にメールを送る場合**：TOに自身のアドレスを入力し、BCCで複数相手のアドレスを指定する。
- **メール誤送信への事前対策**：現場でメールアドレスが支給されたら、設定で「メール送信前に内容の確認を促すアドオン」を利用できるようにする。宛先アドレスは必ず確認。
- **重要な情報や個人情報を送信する場合**：本文に記入せずファイルを添付する形で行う。重要な情報や個人情報を添付ファイルに記載する場合は、「パスワードの設定、またはパスワード付きのZIPファイルにして暗号化」する。パスワードは「先方とあらかじめ決めておく、または電話で知らせるなどパスワードが傍受されないよう配慮」する。

### 5. 業務中の電子メール受信に関するルール（標的型攻撃メール対策）
- **ウイルス感染防止の注意**：以下の内容に複数合致する場合、安易に添付ファイルを開いたり、リンクを参照したりしない。
  - メールのテーマ：知らない人からのメールでURLや添付ファイルを開かざるを得ない内容、心当たりのない興味をそそられる内容、届いたことのない公的機関からのお知らせ、組織全体への案内、心当たりのない決済や配送通知（英文が多い）、IDやパスワードなどの入力を要求するメール。
  - 差出人のメールアドレス：フリーメールアドレスからの送信、差出人のメールアドレスと署名のアドレスが異なる。
  - メールの本文：日本語の言い回しが不自然、日本語では使用されない漢字（繁体字、簡体字）が使われている、実在する名称を一部に含むURLが記載されている、表示されているURL（アンカーテキスト）と実際のリンク先URLが異なる、署名の内容が誤っている。
  - 添付ファイル：ファイルが添付されている、実行形式ファイル（exe / scr / cplなど）が添付されている、ショートカットファイル（lnkなど）が添付されている、アイコンが偽装されている、ファイル拡張子が偽装されている。

### 6. 業務における基本ルール
- **クリアデスク・クリアスクリーンの徹底**：スマートフォン、携帯電話、重要書類、USBメモリ、小型ハードディスク、CD等の電子媒体は業務利用時以外は机上に放置せず、クリアデスクを徹底する。
- **離席時の画面ロック**：以下のいずれかによりパソコンの画面をロックし、クリアスクリーンを徹底する。
  - スクリーンセーバー起動時間を5分以内に設定し、パスワードを設定する。
  - スリープ起動時間を5分以内に設定し、解除時のパスワード保護を設定する。
  - ［Windows］＋［L］キーを押してコンピュータをロックする。
- **退社時・未使用時の保管**：現場支給のノートパソコン、USBメモリ、小型ハードディスク、CD等の電子媒体及び重要書類を机の引き出しに保管し、施錠する。
- **重要情報の持ち出し禁止（原則）**：原則として、現場外への重要情報の持ち出しは一切禁止。「現場社員等からの依頼でやむを得ず持ち出す場合」は、以下の徹底が求められる。
  - 電子媒体はケースに入れ、USBメモリはタグ、ストラップ、鈴などを付け紛失を防止する。
  - 書類はひも付き封筒に入れる。
  - ノートパソコンはログインパスワードを設定する。
  - 電子データはファイル暗号化、またはUSBメモリ暗号化機能により暗号化する。
  - 携行時の注意：電車内では網棚に置かない、自動車内には保管しない、作業中離席する場合は携行する、他者から覗き見できない状態で扱う。
  - その他貸与物についても細心の注意を払う。
- **リモート作業に関するルール**：リモート作業実施前には必ず「リモート作業に関する誓約書」の締結が必要。誓約書に心当たりがない場合は、レバテック担当営業へ問い合わせる。以下の場合は、一時的な場合でも事前にレバテック担当営業へ相談する。
  - 弊社に登録している住所以外で作業する可能性がある場合
  - 転居する場合
  - 国外での作業を希望する場合（認可していない企業もあるため、検討段階で必ず相談）
  - 常駐からリモート作業へ変更になる場合
  - 現場や自宅以外の、カフェやコワーキングスペース等、公共施設での作業は原則禁止。

### 7. 私用情報機器の利用/持込に関するルール
- **私有パソコンの利用/持込**：「現場内へ無断で持ち込むことを禁止する。」
- **業務利用を禁止**（業務利用には別途契約が必要）
- **正式な手続きを踏んで業務利用を行う場合**：ウイルス対策ソフトやアプリケーションソフトはレバテックまたは現場社員担当が指定したものを導入し、許可を得て利用。業務終了後、業務用データは指定ツールで完全に消去。
- **スマートフォン、タブレット端末、携帯電話などの記憶・通信機能を備えた機器**：「レバテックとの連絡を除き業務利用を禁止する。」現場社内パソコンへの接続を禁止。業務用データの保存を禁止。
- **USBメモリ、外付けHDDなどの記憶機能を備えた機器・媒体**：「私有物の現場利用を禁止する。」利用が必要な場合は現場社員の担当の許可を得て利用。私物を業務利用した場合、業務終了後、業務用データを現場社員担当の指定ツールで完全に消去。

### 8. 現場からの貸与物に関するルール
- **貸与物の確認と保管**：貸与物の種類・数の確認を必ず行い、梱包資材含めまとめて保管する。
- **紛失・破損防止の注意**：貸与物は貴重品であるため、紛失や破損しないように注意して扱う。同居家族や同居人がいる場合、厳重に保管し、自身以外に触れられないようにする。過去に紛失や破損により企業から損害賠償を求められたケースがあるため注意。
- **紛失時の緊急連絡**：万が一貸与物を一部でも紛失した場合は、「現場社員のリーダーとレバテック担当に至急連絡」すること。
- **返却義務**：契約終了後は速やかに全ての貸与物を返却すること。
- **特定の貸与物に関する注意**：
  - パソコン：「現場以外へ無断で持ち出しすることを禁止する。」（飲み会等、作業場所以外への持ち出しも禁止）。
  - スマートフォン、タブレット端末、携帯電話など：「現場以外へ無断で持ち出しすることを禁止する。」（飲み会など業務以外の場所への持ち出しも禁止）。スマートフォン/携帯電話は、首から下げる等し自身の身につけた状態にすること。
  - セキュリティカード：企業の情報資産となるため、取り扱いには十分に気を付けること。（例：作業終了後は落下による紛失を防ぐ為、カバンの奥に入れ厳重に管理し、不用意に持ち歩かないようにする）。

## 結論
本ハンドブックは、レバテック経由で業務を行うすべての作業従事者にとって、情報セキュリティを確保するための包括的なガイドラインを提供しています。特に、情報セキュリティ事故の定義とその発生時の報告フロー、業務中のインターネット・メール利用、重要情報の取り扱い、私用情報機器の利用、貸与物の管理に関する詳細な指示が示されています。偶発的な行動も情報セキュリティ事故につながる可能性があるという点が強調されており、作業従事者には、常に規則を遵守し、不明点があれば自己判断せずに現場社員やレバテック担当者に相談することが強く求められています。これらのルールを遵守することで、情報セキュリティリスクを最小限に抑え、安全な業務遂行を可能にすることが期待されます。
