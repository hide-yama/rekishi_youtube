## 使用方法メモ

### タスクの状態マーク
- `- [ ]` : 未着手
- `- [~]` : 作業中（または🔄マーク）
- `- [x]` : 完了

### 優先度アイコン
- 🔴 : 高優先度
- 🟡 : 中優先度  
- 🟢 : 低優先度

### 同期コマンド
```bash
# 今日のタスクを生成
node extract.js --date 2025-05-25 > daily/2025-05-25.md

# 変更をtasks.ymlに同期
node sync.js --file daily/2025-05-25.md
```
